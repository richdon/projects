Introduction¶

In this analysis of gun registration data vs U.S Census Beaureau data, the gun data was found on GitHub but originally extracted from the FBI's National Instant Criminal Background Check System (NCIS). The U.S Census data was found on census.gov.

The census data provided is for the years 2015 and 2016, meaning that some attributes reflect observations from 2015 and others from 2016, for 50 states. The gun registration data has observations indexed by month from 1998 - 2017, for 55 states/territories.

The attributes being used from the census data are going to be based on population demographics, employment, income and povery, and highest level of completed education by state. The attributes for the gun data will be based on the type of gun being registered and if the sale was private. The estimated sale of guns will be calculated by using this equation: (handgun check + long gun check + 2 multple gun check) 1.1 . This equation is suggested in the Small Arms Survey by Jurgen Brauer, a professor at Georgia Regents University " As approximations go, one may then add handgun checks, plus long gun checks, plus two multiple checks (at least one handgun and one long gun), and augment the resulting number by a factor of 1.1, termed here the multiple gun sales factor (MGSF)".

The questions we will be posing are:

  1. which states have the highest total gun sales?
  2. Are gun sales growing or declining?
  3. what are the characteristics of states with higher hand gun purchases and those with higher long gun purchases?
  4. What are the relationships between various census data to private gun sales and regular gun sales?

    5. We will be answering questions 1 and 2 by analyzing only the gun dataset. For the questions 3,4 we will compare both data sets. For this we need to change the shape of the census data. And for the gun data we will aggregate and take the average values of each attribute for the years 2015 and 2016, because the census data attributes reflect both of those years.

# function used to select the row indexes for census data
def rows(index):        
    if index in range(2,13) or index == 15 or index == 17 or index == 18:
        return True
    elif index in range(20,35)  or index in range (37,39) or index in range(40,48) or index == 49:
        return True
    elif index in range(51,86):
        return True
    else:
        return False

# load census dataframe
